Support

SMSLib Blog

Important news and other announcements about new releases, bugs/bug fixes, and general chitchat about the library are posted on the SMSLib Blog.

Support Groups

There are two support groups, one for the original java SMSLib library and one for the SMSLib for .NET port. Please use the appropriate one for your questions and feedback!

SMSLib for Java

The SMSLib for Java User Group is the support group for the original SMSLib java library.

SMSLib for .NET

The SMSLib for .NET User Group is a new support group which will be dedicated to the SMSLib for .NET platform port.

Issue Tracker

The SMSLib Issue Tracker is also available for reporting issues, bugs or filing enhancement requests. Tracker activity can be monitored from the Tracker ATOM feed. Alternatively, you can subscribe to the SMSLib Project Updates Group.

Repository

SMSLib Subversion repository is currently hosted on Google Code.

You can checkout the currently developed SMSLib version (i.e. the "trunk") from here.

Subversion activity can be monitored from the Subversion ATOM feed. Alternatively, you can subscribe to the SMSLib Project Updates Group.

Please note that the repository snapshots may not always work properly. We try our best to commit codes that works, but don't forget that the repo is for development... If you are interested in stable versions, you are advised to following the published releases.
