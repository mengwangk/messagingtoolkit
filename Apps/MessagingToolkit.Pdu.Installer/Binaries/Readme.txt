README
======

Get the related libraries from the respective folders

WinForms
==============
Contains the followings libraries compiled for different environments for WinForms applications.
- Any CPU 
- x86
- x64

Portable
==============
Contains the portable library for use in Silverlight, Windows Phone 7 and above, Windows Store App, .NET 4.0 and above.


.NET 3.5
========
Contains libraries for .NET 3.5 runtime for WinForms.
- Any CPU 
- x86
- x64
- Itanium


Windows 8
==========
Contains libraries for Windows 8
- Any CPU 
- x86
- x64
- ARM
