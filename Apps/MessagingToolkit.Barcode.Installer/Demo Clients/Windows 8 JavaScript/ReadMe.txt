Installation
============

1. Go to "MessagingToolkit.Barcode.Windows8.JavaScriptDemo_1.0.0.0_AnyCPU_Test" folder.
2. Right click on "Add-AppDevPackage.ps1" and select "Run with PowerShell".
3. A screen is popped out to install the application.
4. After the installation, you should be able to run the application from the Start Menu.
