This package contains examples of how to use the UI and RMS toolkits.

This MIDlet ( {@link com.nextel.examples.rms.SalaryKeeper} ) is a simple example of how to use the RMS toolkit, as well as examples of the following UI components:

  • {@link com.nextel.ui.OScrollList}
  • {@link com.nextel.ui.OTextField}
  • {@link com.nextel.ui.OPushButton}
For a more detailed example of using all 3 ( UI, RMS and networking ) toolkits, check out the {@link com.nextel.examples.expensepad.ExpensePad} MIDlet.
