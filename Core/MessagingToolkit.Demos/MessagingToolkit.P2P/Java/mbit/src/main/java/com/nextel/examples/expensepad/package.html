This package contains examples of how to use the RMS, UI and networking toolkits.

This application ( {@link com.nextel.examples.expensepad.ExpensePad} ) was one of the original MIDlets that was included with the Nextel J2ME phones when they were released. This version has been updated to include synchronization of records to a remote servlet.

The main features are:

  • Ability to add, update, delete and query for expense records, using the UI and RMS toolkits.
  • Ability to upload and download records to a remote servlet, using the networking toolkit.

For another example of using the RMS toolkit, see the {@link com.nextel.examples.rms.SalaryKeeper}.
